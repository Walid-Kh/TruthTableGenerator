    <h2 class="text-center mt-3 text-white">Truth Table Generator</h2>
    <p class="text-white text-center">
      This tool generates truth tables for logical expressions.
        variables are considered single character and they can be any latin character capitalized or not,
        parentheses are supported.
        the supported Operations are as follows:
    <ul class="list-unstyled text-white text-center m-0">
      <li><span class="font-weight-bold"> ! </span>NOT operator</li>
      <li><span class="font-weight-bold"> & </span>AND operator</li>
      <li><span class="font-weight-bold"> | </span>OR operator</li>
    </ul>
    </p>
